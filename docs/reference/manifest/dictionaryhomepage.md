# <a name="dictionaryhomepage-element"></a>DictionaryHomePage 要素

辞書のホームページの URL を指定します。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
<DictionaryHomePage DefaultValue="URL" />
```

## <a name="contained-in"></a>含まれるもの

[辞書](dictionary.md)
