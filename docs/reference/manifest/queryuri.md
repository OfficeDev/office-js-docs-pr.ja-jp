# <a name="queryuri-element"></a>QueryUri 要素

辞書クエリ サービスのエンドポイントの URL を指定します。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
<QueryUri DefaultValue="string" />
```
