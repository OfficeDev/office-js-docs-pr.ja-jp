# <a name="progid-element"></a>ProgId 要素

Web アドインの作業ウィンドウ用の対応する COM アドインのプログラム id を指定します。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
    <ProgId>string</ProgId>  
```
