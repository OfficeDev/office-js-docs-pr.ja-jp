# <a name="formsettings-element"></a>FormSettings 要素

メール アドインのソースの場所と制御の設定を指定します。

**アドインの種類:** メール

## <a name="syntax"></a>構文

```XML
<FormSettings>
   ...
</FormSettings>
```
