# <a name="targetdialects-element"></a>TargetDialects 要素

地域言語または、この辞書でサポートされる言語を定義します。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
<TargetDialects>
   ...
</TargetDialects>
```
